# T6 Asset Browser

[![GitHub release](https://img.shields.io/github/v/release/maxvanasten/t6-asset-browser)](https://github.com/maxvanasten/t6-asset-browser/releases)
[![Tests](https://img.shields.io/badge/tests-passing-brightgreen)]()

Browse, search, and export Call of Duty: Black Ops 2 (T6) assets from FastFiles. Generate weapon lists for your GSC mods.

## Features

- **Interactive TUI** - Browse assets with a beautiful terminal UI (vim-like keybindings)
- **Parse FastFiles** - Read zone/*.ff files directly (requires OpenAssetTools)
- **Export Weapons** - Generate complete weapon lists for each map
- **Export Perks** - Find all available perks on each map
- **GSC Generation** - Auto-generate GSC arrays for maxlib.gsc
- **Multiple Formats** - Export as GSC, JSON, CSV, or plain text
- **Caching** - Speed up subsequent runs with built-in cache
- **Cross-Platform** - Works on Linux, macOS, and Windows

## Prerequisites

You must have **OpenAssetTools** installed.

**Installation:**
1. Download from https://github.com/Laupetin/OpenAssetTools/releases
2. Or build from source: https://github.com/Laupetin/OpenAssetTools#building-oat
3. Ensure `OAT.Unlinker` is in your PATH

Without OAT, the tool will attempt built-in decryption which may not work with all FastFiles.

## Quick Start

```bash
# Launch interactive TUI
$ t6-assets -tui

# Index all FastFiles
$ t6-assets -cmd=index

# Export weapons from Origins as GSC
$ t6-assets -cmd=export -map=zm_tomb -type=weapon -format=gsc

# Search for raygun variants
$ t6-assets -cmd=search -pattern=raygun
```

## Interactive TUI Mode

The TUI provides an intuitive visual interface for browsing and searching assets.

### Launching the TUI

```bash
t6-assets -tui
```

### TUI Controls

The TUI features vim-like modal editing:

**Query Builder - NORMAL Mode:**
- `i`, `I` - Enter INSERT mode to edit the current field
- `j` or `↓` - Navigate to next field
- `k` or `↑` - Navigate to previous field
- `Tab` / `Shift+Tab` - Alternative navigation
- `Enter` - Execute query
- `Ctrl+L` - Clear all fields
- `?` or `h` - Show help
- `q` or `Ctrl+C` - Quit

**Query Builder - INSERT Mode:**
- Type freely to enter text
- `Esc` - Return to NORMAL mode

**Results Screen:**
- `j` or `↓` - Navigate down
- `k` or `↑` - Navigate up
- `g` - Go to first result
- `G` - Go to last result
- `Ctrl+D` - Half page down
- `Ctrl+U` - Half page up
- `/` - Search within results
- `n`/`N` - Next/previous search match
- `y` - Copy current item name
- `b` or `Esc` - Back to query builder
- `?` or `h` - Show help
- `q` or `Ctrl+C` - Quit

### TUI Query Configuration

Configure your query using the following fields:

| Field | Description | Example |
|-------|-------------|---------|
| `Command` | Operation to perform | `list`, `search` |
| `Map` | Comma-separated map names | `zm_tomb, zm_prison` |
| `Type` | Asset type filter | `weapon`, `perk` |
| `Pattern` | Search patterns (comma = AND, ! = exclude) | `upgraded, !staff` |
| `Format` | Export format (export mode) | `gsc`, `json`, `csv` |
| `Output` | Output file path | `weapons.txt` |

### Example 2b: List with Pattern Filter

```bash
$ t6-assets -cmd=list -map=zm_tomb -type=weapon -pattern=raygun

[weapon] ray_gun_zm (from zm_tomb.ff)
[weapon] ray_gun_upgraded_zm (from zm_tomb.ff)
[weapon] raygun_mark2_zm (from zm_tomb.ff)
[weapon] raygun_mark2_upgraded_zm (from zm_tomb.ff)

```

### Example 2c: List Excluding Upgraded Weapons (Anti-Pattern)

```bash
$ t6-assets -cmd=list -map=zm_tomb -type=weapon -pattern='!upgraded'

[weapon] hamr_zm (from zm_tomb.ff)
[weapon] galil_zm (from zm_tomb.ff)
[weapon] ak74u_zm (from zm_tomb.ff)
[weapon] mp40_zm (from zm_tomb.ff)
...

Total: 48 assets (excluding upgraded variants)
```

**Note:** Use single quotes around anti-patterns to prevent bash from interpreting `!` as history expansion.

### Example 2d: List with Multiple Patterns (AND Logic)

Use comma-separated patterns to match assets that contain **all** patterns:

```bash
$ t6-assets -cmd=list -map=zm_tomb -type=weapon -pattern='raygun,upgraded'

[weapon] ray_gun_upgraded_zm (from zm_tomb.ff)
[weapon] raygun_mark2_upgraded_zm (from zm_tomb.ff)

Total: 2 assets (weapons with both 'raygun' AND 'upgraded' in name)
```

### Example 2e: Combined Include and Exclude Patterns

Mix positive and negative patterns to fine-tune results:

```bash
$ t6-assets -cmd=list -map=zm_tomb -type=weapon -pattern='upgraded,!staff'

[weapon] ray_gun_upgraded_zm (from zm_tomb.ff)
[weapon] raygun_mark2_upgraded_zm (from zm_tomb.ff)
[weapon] hamr_upgraded_zm (from zm_tomb.ff)
...

Total: 24 assets (upgraded weapons but NOT staff weapons)
```

## Available Commands

- `index` - Index all FastFiles and build asset database
- `list` - List assets (optionally filter by map and type)
- `search` - Search for assets by name pattern
- `export` - Export assets to various formats
- `tui` (flag `-tui`) - Launch interactive terminal UI

## Available Asset Types

| Type | Description | Example |
|------|-------------|---------|
| `weapon` | Weapons | `ak47_zm`, `ray_gun_zm` |
| `xmodel` | 3D models | `viewmodel_ak47_mp` |
| `perk` | Perks/specialties | `specialty_armorvest` |
| `material` | Materials | `mc/mtl_ak47_body` |
| `image` | Textures | `i_ak47_col` |

## Command Line Flags

| Flag | Description | Default |
|------|-------------|---------|
| `-tui` | Launch interactive terminal UI | false |
| `-zone-dir` | Path to zone directory | Auto-detect |
| `-cmd` | Command to run | index |
| `-map` | Map name filter | (none) |
| `-type` | Asset type filter | (none) |
| `-pattern` | Pattern(s) to filter asset names. Use comma for AND logic, `!` prefix to exclude | (none) |
| `-i` | Case-insensitive pattern matching | false |
| `-wildcard` | Use wildcards (* and ?) in pattern | false |
| `-format` | Export format | plain |
| `-output` | Output file | stdout |
| `-cache` | Use caching | true |
| `-clear-cache` | Clear cache before running | false |

## Troubleshooting

### "OpenAssetTools not found"

Install OAT and ensure it's in your PATH:
```bash
# Verify OAT is available
which OAT.Unlinker
# or on Windows: where OAT.Unlinker
```

### "No FastFiles found"

Check your zone directory:
```bash
ls ~/.steam/steam/steamapps/common/Call\ of\ Duty\ Black\ Ops\ II/zone/all/*.ff
```

### Empty results after indexing

This usually means OAT isn't working. Verify:
1. OAT.Unlinker is installed and in PATH
2. Your game files are valid
3. Try clearing cache: `t6-assets -clear-cache`

### Cache issues

Clear the cache:
```bash
t6-assets -clear-cache
```

Cache is stored in:
- Linux/macOS: `/tmp/t6-asset-browser-cache/`
- Windows: `%TEMP%\t6-asset-browser-cache\`
